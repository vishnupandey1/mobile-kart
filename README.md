# Mobile E-KART
This is a modern, responsive mobile-kart.

Check demo at: https://nervous-babbage-981795.netlify.com/

## Installation

    $ https://github.com/vishnupandey1/mobile-kart.git
    $ cd mobile-kart
    $ yarn install

To Run Test Suite:

    $ yarn test

To Start Server:

    $ yarn start

To Visit E-KART:

    http://localhost:3000
